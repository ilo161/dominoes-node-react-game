# dominoes-node-react-game
Dominoes game for multiple players implemented with ReactJS and nodejs.

## How to play and work in development mode
### 1) Nodejs backend
1. `cd backend`
2. `npm install`
3. `npm run start-watch`
4. Verify: `curl localhost:8000/health`

### 2) React frontend
1. `cd frontend`
2. `npm install`
3. `npm run start`
